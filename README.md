### Disaster Response Simulation: Ambulance Allocation

This project was completed for the course **Simulation - Modeling, Analysis and Applications**.

The project implements a discrete-event simulation using SimPy to model ambulance allocation after a mass-casualty disaster. It compares different ambulance allocation strategies across three disaster sites and evaluates survival rates using multiple replications, confidence intervals, optimization, and sensitivity analysis.


## Research Question

"How should 24 ambulances be allocated across 3 disaster sites to maximize patient survival rate?"


## Main Methods

- Discrete-event simulation
- SimPy process modeling
- Multiple replications
- 95% confidence intervals
- Exhaustive search optimization
- Sensitivity analysis
- Output analysis and visualization


## Tools
- Python
- SimPy
- pandas
- NumPy
- matplotlib
- seaborn


## Files

- `disaster_response_simulation.ipynb` - main notebook
- `report.pdf` - final project report

## How to Run

Open `disaster_response_simulation.ipynb` and install the packages listed in `requirements.txt` using:
```bash
pip install -r requirements.txt
```
then run each notebook cell from top to bottom.
